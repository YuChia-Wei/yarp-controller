## 系統用途、目的

- 管理 Yarp 設定檔案
- 管理使用 Yarp 所建立的服務

## 技術與架構

此專案預期利用以下技術、技能進行建置與開發，核心目標是在開發初期讓我練習這些內容。

1. 六角架構(Hexagonal architecture) / 乾淨架構(Clean architecture)
2. DDD
3. .net 8
4. RESTful WebApi Design
5. MQTT (or RabbitMQ and other message queue system)
